## Deploy Online

This site is hosted on **GitHub Pages** from the `gh-pages` branch.

### Enable the site (one time only)

1. Open **[github.com/sagarkadam7/sp/settings/pages](https://github.com/sagarkadam7/sp/settings/pages)**
2. Under **Build and deployment**, set **Source** to **Deploy from a branch**
3. Choose **Branch:** `gh-pages` and **Folder:** `/ (root)`
4. Click **Save**
5. Wait 1–2 minutes, then open **https://sagarkadam7.github.io/sp/**

Future updates pushed to `main` deploy automatically via GitHub Actions.
