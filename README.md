**https://sagarkadam7.github.io/sp/**

Repository: [github.com/sagarkadam7/sp](https://github.com/sagarkadam7/sp)

- **Hero section** — Introduction and call-to-action

## View the Website

Open `index.html` in any web browser:

1. Double-click `index.html`, or
2. Right-click → Open with → Chrome / Edge / Firefox

## Customize Before Sharing

| Placeholder | Location |
|-------------|----------|
| Phone number | Contact section, hero, about card (`+91 95185 47622`) |

## Deploy Online

This site is hosted on **GitHub Pages**. After pushing to `main`, it deploys automatically.

If the live link shows 404 the first time, enable Pages once:

1. Open [github.com/sagarkadam7/sp/settings/pages](https://github.com/sagarkadam7/sp/settings/pages)
2. Under **Build and deployment**, set **Source** to **GitHub Actions**
3. Wait 1–2 minutes, then open **https://sagarkadam7.github.io/sp/**

Alternative free hosts: [Netlify](https://netlify.com) or [Vercel](https://vercel.com)
